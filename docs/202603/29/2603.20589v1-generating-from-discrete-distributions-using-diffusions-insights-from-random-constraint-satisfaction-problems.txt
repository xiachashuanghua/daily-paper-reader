Abstract 

Generating data from discrete distributions is important for a number of application domains including text, tabular data, and genomic data. Several groups have recently used random k-satisfiability ( k-SAT) as a synthetic benchmark for new generative techniques. In this paper, we show that fundamental insights from the theory of random constraint satisfaction problems have observable implications (sometime contradicting intuition) on the behavior of generative techniques on such benchmarks. More precisely, we study the problem of generating a uniformly random solution of a given (random) k-SAT or k-XORSAT formula. Among other findings, we observe that: (i) Continuous diffusions outperform masked discrete diffusions; ( ii ) Learned diffusions can match the theoretical ‘ideal’ accuracy; ( iii ) Smart ordering of the variables can significantly improve accuracy, although not following popular heuristics. 

# 1 Introduction 

Since the introduction of denoising diffusions [SDWMG15, SE19, HJA20, SSDK +21], an impressive number of variants and generalizations have emerged. An important motivation for generalizing the original framework has been to develop diffusions-style generative techniques for discrete data, most notably language and tabular data [AJH +21, HNJ +21, SHW +24]. Despite substantial amount of work, the relative merits of different techniques for generat-ing discrete data are poorly understood. This is particularly true for tabular data generation [LQBvdS23, ZZS +24, SXH +24], an application domain that lacks a universally representative set of benchmarks. Several groups have recently used random k-satisfiability ( k-SAT) as a synthetic benchmark for discrete data generation [SLB +19, FK22, YGG +25, AYN +25]. However, the choice of the setting and metrics in these works is somewhat arbitrary, and not guided by fundamental principles. In particular, there is no notion of what is the target accuracy the generative model should strive for, or how to choose the parameters of the k-SAT distribution. The present work builds upon the wealth of mathematical understanding of random CSPs that has been developed over the last twenty years, through an interdisciplinary effort across computer science, statistical physics, and probability theory, see [MPZ02, KMRT +07, DSS22, ACOGM20] for a few pointers. The unifying theoretical concepts beyond these advances were rooted in spin glass theory [MPV87, MM09]. Recently, methods from this literature were used for construct algo-rithms that provably sample from high-dimensional and highly multi-modal Gibbs measures arising from spin glass models or random CSPs [EAMS22, MW23, HMP24, GDKZ24, MW25] (see also [MRTS07] for an earlier example). These algorithms implement denoising diffusions (possibly dis-crete diffusions [MRTS07, GDKZ24]) with the difference that the denoiser (equivalently, the score) is not computed from data, but is instead motivated by the structure of the target distribution. Namely, these works use (variants of) loopy belief propagation (BP) which, for large random in-stances, is expected to provide a near-optimal denoiser (among polynomial-time algorithms) and hence diffusion sampler. In this paper, we take this line of work one step forward, by investigating learned diffusions and comparing them with the behavior predicted from the theory for BP-based diffusions. An instance of a CSP is determined by a set of constraints over the vector x ∈ X N , with X afinite alphabet (throughout the paper we will consider binary CSPs, i.e. X = {+1 , −1}). A solution of a specific CSP instance is a vector x that satisfies all the constraints. We attempt to generate samples from the uniform measure over solutions of a given random CSP, using both standard denoising diffusions and masked discrete diffusions. For each of these two approaches and each of two families of random CSPs ( k-SAT and k-XORSAT), we study the behavior generative methods of several learned or BP-based denoisers. (Throughout, we use the term ‘denoiser’ broadly: in the case of masked discrete diffusions this is the conditional probability model.) We succinctly summarize some of our findings below, and refer to Figure 1 for an illustration of some of these results: 21. Continuous diffusions outperform masked discrete diffusions. One of the main motivation for the introduction of discrete diffusions was the idea that they are better adapted to discrete data [HNJ +21, AJH +21]. We show that this is not the case for random CSPs, in agreement with theoretical asymptotic predictions [MRTS07, GDKZ24]. 

2. Ideal target accuracy from random CSP theory. We mostly focus on ‘local’ denoisers, i.e. denoisers that estimate each variable using a bounded radius neighborhood of that variable in the graph G which represent the CSP instance. Within the class of r-local denoisers we provide a prediction for the ideal ‘local’ denoiser, i.e. the denoiser that minimizes the score matching objective in the large size, large sample size limit. 

3. Learned diffusion match theoretical target. We show empirically that learned local denoisers can match the behavior of the ideal denoisers at the previous point. 

4. Accuracy is improved by smart ordering (continuous and discrete). We study a more general class of generative methods which treats different coordinates xi in a different way, depending of their relations to other variables, as specified by the graph G. In the case of masked discrete diffusions, this corresponds to masking/unmasking variables in a smart order. In the case of continuous diffusions, this corresponds to adding noise with different variances to different variables, and denoising correspondingly. We show that one can construct orderings that significantly improve the success rate of both discrete and continuous diffusions.

Satisfiability and dynamical phase transitions. As N, M → ∞ , with constant number of constraints per variable M/N = α, the space of solutions of random k-SAT/XORSAT undergoes certain phase transitions that corresponds to qualitative changes of its structure at well-defined critical values 4 of α. Two phase transitions are particularly relevant for what follows (their location differ between SAT and XORSAT, but we omit this dependence unless it is useful to distinguish): (1) The satisfiability phase transition at αs(k): for α < α s(k) a random instance G has solutions with high probability (whp, i.e. with probability converging to one as N, M → ∞ ); for α > α s(k)it does not, always whp. (Further, for α < α s(k), the number of solutions is exponentially large in 

N .) (2) The dynamical (or, shattering) phase transition at αd(k): for α < α d(k) the set of solutions 

S ⊆ {± 1}N is whp ‘well connected’ while for α > α d(k) it can be partitioned into an exponential number of components S = ∪Lj=1 Sj , where any two distinct components have Hamming distance 

d(Si, Sj ) ≥ cN .The dynamical phase transition is particularly important for us because it is expected that no polynomial time algorithm can sample k-SAT solutions for α ∈ (αd(k), α s(k)) [KMRT +07, BH22]. For k-XORSAT sampling is always tractable exploiting the linear-algebraic structure, but local algorithms are believed to fail for α ∈ (αd(k), α s(k)). For these reasons, we will focus on sampling solutions of random formulas in the regime α ∈

(0 , α d(k)).

# 3 Main results 

In this section we present our empirical and theoretical results. We defer technical details to the appendices. We emphasize that BP denoisers do not require training. For NN denoisers, we train on solutions/noisy observation pairs x(s), y(s,ℓ ). In this case of XORSAT it is easy to generate uniform solutions using linear-algebraic methods. However, we find it more effective to use a curriculum, whereby the initial phase of training uses a biased distribution over solutions. The difficulty of training a denoiser for XORSAT solutions is possibly related to barriers to learn XOR (parity) functions via gradient-based methods [AAM23]. For SAT, we experimented with several heuristics to generate solutions, as discussed in Appendix B. We will typically compare various generative methods in terms of the probability that they generate actual solutions (‘success rate’). This is a relatively strict criterion since a single incorrect variable can result in a failure (part of previous work considered the fraction of satisfied constraints, which is a weaker metric.) Of course, high success rate does not guarantee the generated solutions to be uniformly distributed. We will test approximate uniformity in Section 3.4. The reversed-leaf ordering is based on the following ‘leaf removal’ procedure that starts from a XORSAT formula, described by a factor graph G = ( V, F, E ), and progressively removes variables to obtain an empty formula [DM02]. Leaf removal recursively selects one variable that appears only in one constraint (a degree-one variable node in the factor graph), removes that variable and the corresponding constraint. After j successful steps, the original formula ( N variable with M

constraints) is reduced to a formula with N − j variables and M − j constraints, call it Gj (whereby 

G0 = G). The rationale is that at each step j, if the formula Gj can be solved, then any solution can also be extended to a solution of the formula Gj−1.The procedure can end after M , steps, hence resulting in a ‘trivial’ formula with N −M variables and no constraints, or halt before, with a system in which each variable appears in two or more constraints. It is known that the first event occurs with high probability for α < α d(k) and the latter for α > α d(k) [DM02, DM08]. The reversed-leaf ordering corresponds to ordering variables 11 in reverse order with respect to how they are visited by leaf removal. We note that this ordering is not unique. # 4 Discussion 

This paper demonstrated that the uniform measure over solutions of random CSP formulas can be a useful benchmark for generative methods, which can be challenging for two reasons: ( i) Large 13 formula close or beyond the dynamical phase transitions αd(k); ( ii ) CSPs with special symmetry properties such as XORSAT, which might lead to difficulty in training. An important feature of using random CSPs as benchmark is that theory-motivated methods such as BP provide a very strong baseline to compare to. A number of questions are left open, most notably achieving better uniformity in generation, and optimal ordering strategies.

In this section, we formally define the architecture of the neural network employed to learn a denoiser from data. Consider a factor graph G = ( V, F, E ) characterizing a CSP instance, and the noisy information about a solution of that instance, Y . In continuous denoising diffusions, 

Y is obtained by adding Gaussian noise to the true solution x. In masked discrete diffusions it is obtained by masking some randomly selected variables in x (more details on the diffusion are provided in Appendix B.1). The neural network for a radius r denoiser makes a prediction of the true value of the i-th variable in the solution xi by taking into account the noisy values of the r-hop neighbours of variable i in G.Since our architecture is recursive, we will first describe the architecture for r = 1 and sub-sequently describe the construction of the radius-r network from a radius-r − 1 network.
